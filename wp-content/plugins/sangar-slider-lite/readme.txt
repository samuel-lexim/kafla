== Description ==
Sangar Slider is a high quality responsive slider plugin for WordPress. With smooth animation, layered slide, touch and swipe support, easy to use and truly responsive. Packed with premium skins and templates, and also easy to create your very own custom template for your slider. It's full-featured, modern, yet easy to use. Sangar Slider is an awesome layer slider as well, with the ability to adding any HTML contents (texts, images, â€¦) in layers. You can even create your very-own responsive slider theme and template using its extensive API system.

[youtube http://www.youtube.com/watch?v=SKcVL9Zff-M]

No coding required. Sangar Slider Premium is equipped with a cutting-edge, powerful yet intuitive and easy to use layer editor. With layer editor, you can create your layer content by drag-n-drop your content. Each element of the layer is editable, including content transition. The slider also comes with eye-catching animation.

**Usage Instruction**

1. Install the plugin.
2. In the Admin Panel, Go to Sangar Slider
3. Add new slide
4. Fill the setting according your need
5. Read Documentation for more responsive slider detailed instructions

== Installation ==

1. Grab the responsive slider plugin from wordpress plugin directory or Upload the sangar-slider folder to the /wp-content/plugins/ directory
2. Activate the plugin
