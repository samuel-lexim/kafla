Create beautiful Slider based on Post or Category. This is an addon of Sangar Slider plugin for WordPress. 

== Description ==
Post Slider is a high quality Sangar Slider addon for WordPress. 
Using Post Slider, you can create beautiful, intiuitive and responsive slider based on your post or category. 
Post Slider is very easy to use, only with few clicks you can make beautiful slider based on your post or category. 
Post Slider is an addon of <a href="http://wordpress.org/plugins/sangar-slider-lite/" >Sangar Slider</a>. You must have Sangar Slider installed to use this plugin


**Features :**

* Fast and Smooth Animation
* Create Slider Based on Post or Category
* Ultra Responsive
* Beautiful, Pre-defined Template
* Integrated Seamlessly with Sangar Slider
* Touch and swipe gesture
* Easy to use

**Usage Instruction**

1. Install Sangar Slider plugin.
1. Install Post Slider plugin.
2. In the Admin Panel, Go to Sangar Slider > Post Slider
3. Add new slide
4. Fill the setting according your need
5. Read Documentation for more detailed instructions


== Installation ==

1. Grab the plugin from from wordpress plugin directory or Upload the post-slider folder to the /wp-content/plugins/ directory
2. Activate the plugin
